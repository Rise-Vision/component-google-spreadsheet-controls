Spreadsheet Controls

An Angular directive used to configure a chosen Google Spreadsheet

Dependencies Google Client JSAPI Google Drive Picker Tooltip UI Bootstrap

Usage

Clicking the Google Drive Picker button will launch the Google Picker modal to which you can then select a spreadsheet.

Upon selecting the spreadsheet, provided it is public and been published, you can then edit several configurations including which worksheet to display. If the spreadsheet is not public or published, the directive displays an error message and a hyperlink to open the document (see Google's documentation on public and published spreadsheets).

The directive requires the use of the spreadsheet attribute which defines an object and instantiates/saves the following properties: docURL, docName, fileId, url, cells, range, sheet, headerRow, refresh.

Optionally add ng-model variable for validation.
